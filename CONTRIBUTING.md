# Contributing

Changes to this repository must follow the [WeianData Engineering Handbook](handbook/README.md).

## Change path

1. Select the task route and operating mode in [`handbook-manifest.json`](handbook/handbook-manifest.json).
2. Open a focused branch and issue or brief appropriate to the mode.
3. Update authoritative Markdown and derived indexes together.
4. Run `python3 handbook/tools/validate_handbook.py`.
5. Submit a pull request with the evidence required by the [Pull Request Standard](handbook/chapters/30-pull-request-standard.md).

Do not include credentials, restricted client data, or confidential client artifacts.
